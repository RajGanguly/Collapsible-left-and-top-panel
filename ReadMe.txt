This code should work smoothly in modern systems and cross-platform mobile applications.

- LeftPanel : Consisting of icons and text as well as the li elements have an example server re-direction url.

- Navbar : Topmost collapsible navbar contains icons and text and have example usage of server re-direct

- Sub-menus: A generic submenu code is included which shows how a particular submenu can be added to a typical li item of the leftPanel upto level 1. Using this, different levels of submenus to the leftPanel can be created.

-Pre-requisites : Jquery JS and CSS files.

- Test: Google Chrome, iOS, Android and Windows ExE
